# CCNIF Data Schema v1.0

## Purpose

This document defines the standard structure of every statistical table
generated by CCNIF.

These schemas act as stable interfaces between modules.

------------------------------------------------------------

# Transcriptomics_Distribution.tsv

Rows
One row per gene

Columns

Gene
GeneID
log2FC
AbsLog2FC
padj
baseMean
Rank
Percentile
