# CCNIF

## Cancer Causal Network Inference Framework

CCNIF is an explainable computational oncology framework for integrating transcriptomic, somatic mutation and clinical datasets to identify candidate molecular drivers of tumor progression.

## Current Dataset

Cancer:
TCGA-LUAD

Matched Patients:
505

Expression Genes:
17,629

Mutation Genes:
16,399

Clinical Variables:
11

## Pipeline

1. Data acquisition
2. RNA-seq preprocessing
3. Clinical harmonization
4. Mutation processing
5. Multi-omics integration
6. Driver prioritization
7. Regulatory network inference
8. Explainable AI
9. Biomarker ranking
10. Publication-ready figures
